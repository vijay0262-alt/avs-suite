## Getting Started

### System Requirements

- Windows 10 (64-bit) or Windows 11
- 2 GB RAM minimum (4 GB recommended)
- 200 MB free disk space
- Administrator privileges for installation

### Installation

1. Download the AVS PC Optimizer installer from the official website.
2. Run the installer and follow the on-screen instructions.
3. Launch AVS PC Optimizer from the Start Menu or Desktop shortcut.

### First Launch

On first launch, the application will:
- Start the backend optimization engine
- Display the Dashboard with system health overview
- Begin monitoring system performance

## Dashboard

The Dashboard provides an at-a-glance view of your system's health.

### Health Score

The health score (0-100) is calculated from:
- CPU usage
- Memory usage
- Disk usage
- Startup items count
- Junk file volume
- Privacy issues

### Health Issues

Issues are categorized by severity:
- **Critical** (red): Immediate action recommended
- **Warning** (yellow): Action suggested
- **Info** (blue): Informational

### Smart Health Scan

Click "Optimize Now" to run a comprehensive system optimization.

## Junk Cleaner

Scans and removes unnecessary files to free up disk space.

### Basic Cleaning (Free)
- Temporary files
- Windows log files
- Recycle Bin contents
- Thumbnail cache

### Deep Cleaning (Pro)
- Browser caches (Chrome, Firefox, Edge)
- Application cache data
- Download history
- System trace files

---

## Registry Cleaner

Scans the Windows Registry for invalid entries and safely removes them.

### Categories
- Missing shared DLLs
- Unused file extensions
- ActiveX/COM issues
- Application paths
- Help files
- Startup entries

### Backup
The Registry Cleaner automatically creates a backup before any changes. Use the "Restore" option to revert if needed.

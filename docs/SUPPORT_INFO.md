## Before Contacting Support

Please gather the following information:
1. **Application version** — Found in About page or Settings > Version
2. **Operating system** — Windows version and build number
3. **Error description** — What happened, what you expected
4. **Steps to reproduce** — Exact steps that trigger the issue
5. **Log file** — Located at `%APPDATA%\@avs\pc-optimizer\logs\main.log`

When reporting bugs, include:
1. Application version and build number
2. Windows version
3. Detailed description of the issue
4. Steps to reproduce
5. Expected vs actual behavior
6. Relevant log entries
